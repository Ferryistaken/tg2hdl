# tg2hdl

A modern documentation site for a tinygrad-to-hardware exploration centered on an INT8 GEMV unit in Amaranth.

## What makes this docs site different now?

- It is built with **MkDocs Material** (modern UI, fast search, great readability).
- It includes **automatic API docs from Python docstrings** using `mkdocstrings`.
- It keeps conceptual guides and generated API reference in one place.
